Tonight · sleep

Let your ring fill the diary

Your Oura ring already measures everything the sleep window needs. A one-time Apple Shortcut hands it to the app, so you don't type four numbers every morning — and it can backfill your whole history at once, which is the only way the window is ready before the peak week.

Why a Shortcut, and not a button

A web app can't read Apple Health (Safari has no Health API), and it can't call Oura directly either — Oura's servers refuse cross-site browser requests. An Apple Shortcut has neither limit, so the Shortcut does the fetching and the app just reads what it hands over. Nothing leaves your phone except the request to your own Oura account.

1Get an Oura token

  1. On any browser, open cloud.ouraring.com/personal-access-tokens and sign in.
  2. Tap Create A New Personal Access Token, give it a name (“Taper”), and copy the long string it shows. You only see it once.

A personal token only reads your own data and you can revoke it on that same page anytime.

2Build the Shortcut

Open the Shortcuts app → + → add these actions in order:

  1. Text — paste your token into it.
  2. Get Contents of URL. Set the URL to
    https://api.ouraring.com/v2/usercollection/sleep?start_date=2026-06-01&end_date=2026-12-31
    Expand Headers and add one: key Authorization, value Bearer followed by the Text from step 1.
  3. Copy to Clipboard — pass it the Contents of URL.

Run it once. Then in the app, open the Tonight card → Import from Oura or Apple Health → paste → Import sleep. The wide date range backfills everything at once; after that you can narrow it.

The app understands Oura's raw response as-is — you don't have to reformat anything. It reads bedtime, wake time, how long you took to fall asleep and time awake, and keys each night to the morning you got up.

3Make it automatic (optional)

To skip even the paste, have the Shortcut open the app with the data attached instead of copying it:

  1. Delete the Copy to Clipboard action.
  2. Add URL Encode — pass it the Contents of URL.
  3. Add Text with:
    YOUR-APP-URL/index.html#sleep= then the URL Encoded Text right after the =.
    Replace YOUR-APP-URL with wherever this app lives (the address in your browser when you use it).
  4. Add Open URLs — pass it that Text.

Now running the Shortcut opens the app and imports silently. Point a Personal Automation (Shortcuts → Automation → time of day, each morning) at it and you never touch it again. For a daily run, set the URL's start_date to yesterday so it only pulls the newest night.

The automatic path depends on your phone opening the installed app rather than Safari, which isn't always reliable. If a morning doesn't land, the paste box from step 2 always works — and the app scrubs the data out of the address bar the moment it reads it, so nothing lingers there.

Apple Health instead of Oura

If you'd rather not make a token, a Shortcut can read the Find Health Samples of type Sleep Analysis that Oura writes into Health, and copy those. It works, but Health stores sleep stages rather than the four tidy numbers, so the app has to reconstruct latency and time-awake from them — a little less exact than reading Oura directly. If you have the token, step 2 is the cleaner path.
